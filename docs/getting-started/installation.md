# Installing continuiti

To install the latest development version use:
```
git clone https://github.com/aai-institute/continuiti.git
cd continuiti
pip install -e .
```

## Dependencies

**continuiti** requires Python>=3.9 and is built on top of
[PyTorch](https://pytorch.org/).
